# Laboratório de Gestão de Configuração Moderna com Git e GitHub

Este projeto foi desenvolvido como parte da disciplina de Engenharia de Software.

## Objetivo

Aplicar conceitos de Gestão de Configuração Moderna utilizando Git e GitHub, incluindo:

- Controle de versão
- Commits padronizados
- Branches
- Merges
- Trabalho colaborativo

## Tecnologias

- Git
- GitHub
- Python
